|       Command        |     Description      |
| :------------------: | :------------------: |
| `vi <file-name> + n` |    转到文件的n行     |
|  `vi <file-name> +`  |      转到文件尾      |
|          0           |         行头         |
|          ^           | 行头第一个可显示字符 |
|          $           |         行尾         |
|          nG          |      转到第n行       |
|          w           |      下一个单词      |
|          b           |      上一个单词      |
|          e           |        单词尾        |
|          I           |       行首插入       |
|          A           |       行尾追加       |
|          d$          |      删除至行尾      |
